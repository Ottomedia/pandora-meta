# Pandora developers tools
`pandora` è il nome dell'insieme di plugin e tema WordPress sviluppati da Ottomedia come CRM della rete Ufficiarredati.it

L'instllazione operativa di `panodra` è [BcOk.it](https://bcok.it)

In questo repository salviamo la documentazione (work in progress) per gli sviluppatori (nel Wiki), script di aiuto, riferimenti, liste "promeoria" ecc.

I singoli plugin e il tema che compongono `pandora` e `BcOk` sono repository privati
